# Saucedemo
Automation Test

## Project Overview

This project contains an **end-to-end automated test suite** for the [Sauce Demo](https://www.saucedemo.com/) e-commerce website. The automation framework is designed to validate the complete purchase journey from login to order completion.


### Objective
Design and implement a comprehensive automated testing solution that demonstrates:
- Test automation design skills
- Clean code organization
- Long-term maintainability
- Industry best practices


### Application Under Test (AUT)
- **URL**: https://www.saucedemo.com/
- **Type**: E-commerce Demo Application
- **Purpose**: Educational testing platform



## Test Scenario
### End-to-End Flow: Place an Order Successfully

The test suite validates a complete user journey:

1. **Login** → User authentication with valid credentials
2. **Browse Products** → View product catalog
3. **Add to Cart** → Select at least two different products
4. **Review Cart** → Verify selected items
5. **Checkout** → Enter shipping information
6. **Verify Pricing** → Validate price calculations
7. **Complete Order** → Finalize purchase
8. **Logout** → End session
